### 教程
1. 进入 [Release 页面](https://github.com/your-repo/releases) 下载最新版本。
2. 准备课表数据：
   - **HNU 同学**: 直接下载 `.xls` 格式课表。
   - **其他学校**: 复制教务系统课表文本。
3. 打开此 [在线格式化工具](https://your-pages-url)，选择你的学校适配器，上传文件进行转换。
   - 工具会自动处理格式、精简地点、生成配置代码。
   - 如果没有你的学校，可以手动参考 `docs/参考格式.txt` 编辑，或贡献适配器。
4. 一键复制生成的文本。
5. 打开应用 <img src="./docs/logo.svg" height="20" align="top">，点击导入，粘贴保存即可！
