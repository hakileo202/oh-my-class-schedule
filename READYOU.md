# READYOU - Oh My Class Schedule

> 不仅仅是一张课表，而是您桌面上的一件艺术品。
> Not just a schedule, but a piece of art on your desktop.

这是一个追求 **极致视觉体验** 与 **无感智能适配** 的跨平台课程表应用。我们相信，每天都要查看无数次的工具，值得被精心雕琢。

## ✨ 眼前一亮的细节

### 1. 🌈 开放的适配器生态 (New!)
- **文件即插件**: 我们重新设计了数据预处理流程。现在，支持一个学校仅需编写一个简单的 `.js` 文件。
- **全自动识别**: 只需要将适配器文件放入 `docs/adapters/`，我们的自动化脚本就会将其注册到 Web 工具中。
- **HNU 专属优化**: 作为首发适配，我们为湖南大学提供了 `.xls` 原生解析支持。拖入教务系统导出的 Excel，不需要任何手动调整，地点自动精简，格式自动对其。

### 2. 🧠 懂你的智能网格
- **今日高亮**: 只有在当前查看的周次确实是“今天”所在的周时，当天的列才会亮起。配合微妙的呼吸光效，让你的注意力自然聚焦。
- **周末自动折叠**: 系统会智能分析当前周的课程。如果您的周六和周日都没有课，它们会自动隐身，将屏幕空间毫无保留地留给工作日。
- **极简表头**: 摒弃了冗余的“星期几”，我们采用了极简的“一、二、...、日”单字表头，清爽干练。

### 3. 🎨 惊艳的视觉呈现
- **玻璃拟态**: 默认采用行业前沿的磨砂玻璃风格，背景中的动态光球在半透明的卡片后缓缓浮动，既有深度又有呼吸感。
- **霓虹深渊**: 专为夜猫子打造的 Cyberpunk 风格，高对比度荧光色，在深夜里熠熠生辉。
- **全屏沉浸**: 无论是在 Windows 桌面还是 Android 手机，应用都会强制占满 `100vh` 高度。每一寸屏幕都被完美利用。

### 4. ⚡ 纯粹的技术架构
- **Rust 后端**: 不再包含任何复杂的业务逻辑，回归最纯粹的高性能文本解析器。稳如磐石，极速响应。
- **Web 前端**: Vue 3 + TypeScript 带来丝滑的交互。逻辑前移，让社区更容易参与贡献。
- **自动化**: 无论是发布版本 (`update_version.py`) 还是更新网页工具 (`update_web_tools.py`)，我们都构建了全自动化的工作流。

## 🤝 如何贡献 (Contributing)

我们非常欢迎您为自己的母校添加适配支持！得益于我们的“文件即插件”架构，这变得异常简单：

1.  **复制模板**: 在 `docs/adapters/` 目录下，找到 `template.js`，将其复制并重命名为 `你的学校.js` (例如 `清华大学.js`)。
2.  **实现逻辑**: 打开新文件，参考 `template.js` 或 `湖南大学.js` 实现 `convert` 方法。你只需要将文件解析为我们定义的文本格式即可。
3.  **自动注册**: 运行自动化脚本更新列表：
    ```bash
    python scripts/update_web_tools.py
    ```
4.  **提交 PR**: 没错，只需编写这一个文件，你的学校就会出现在我们的支持列表中，造福你的所有同学！（从此享受崇敬的目光，也许还能迎来一段迷人的恋情！）
